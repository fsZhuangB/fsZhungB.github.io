C++学习之指针的基本运算

这一篇笔记记录了指针的基本运算，以及它作为左值和右值的用法。

运算符&与*#

&操作符#

要注意，表达式&ch、&cp只能作为右值，不能作为左值（即&ch=xxx），因为虽然cp的值就是&ch的地址值，但是并没有一个明确的地址空间去储存&ch的值。
当然，可以使用一个二级指针(char**)去保存指针cp的地址，从而取到cp地址。

*操作符#

将*cp放在左值和右值的操作是不同的：

  • 放在右值，只会取出其中的值，进行接下来的赋值操作，比如int xx = *cp
  • 放在左值，会取出其中的地址，这样的话可以进行接下来的操作，比如*cp = 3

这个意思是相同的，*号的优先级高于+号，即先进行解引用操作，然后加一，从a变成b。

这个也是同理，放在右值，取的是其中的值，而放在左值，则会取出其中的地址。
