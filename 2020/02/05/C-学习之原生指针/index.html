C++学习之指针基本介绍

最近在重新复习C++， 下面是我学习C++指针的笔记：

变量

首先我们要理解C++中的变量，要知道：
变量的重要信息：

  1. 变量的地址位置
  2. 变量所存的信息
  3. 变量的类型
    这是我们理解指针的基础。

    计算机中的层次储存结构

C++中内存单元内容和地址

指针本身就是一个变量，它储存的是值的地址。对于类型T，T是“到T的指针”类型，一个类型为T的变量能保存一个类型T的对象的地址。如果想要获取该地址，需要使用取地址符号&。

注意，因为引用不是对象，没有实际地址，所以不能定义指向引用的指针。

左值和右值

一般来说，编译器为其单独分配了一块存储空间，可以取到其地址的，就是左值，左值可以放在赋值运算符左侧；

右值指的是数据本身，不能取到其自身地址的，只能放在赋值运算符右边。

具体分析

左值最常见的情况就是函数和数据成员名。

右值则是没有标识符，不可以取地址的表达式，一般也称为“临时对象”

指针数组和数组指针

  1. 指针数组中，里面每个值都是一个指针：T* t[]
  2. 数组指针：该指针指向一个数组：T (*t) []

指针和const

const可以修饰两种：

  1. 一种是const pointer，即指针指向不能改变；（顶层const），同时，它必须初始化。
  2. 另外一种是pointer to const，即指针指向一个常量。（底层const）

野指针

野指针指指向垃圾内存的指针。if等判断对其不起作用，因为没有置为NULL；

一般有三种情况：

  • 指针变量没有初始化。
  • 已经释放不用的指针没有置为NULL，如delete和free之后的指针。
  • 指针操作超越了变量的作用范围。

NULL指针

指一个特殊的指针变量，表示不指向任何东西。如：int* a = NULL

注意：

  • 对一个指针进行间接引用之前，先判断指针是否为NULL！

  • 不知道指针具体指向什么地址之前，可以先将其设置为NULL。
