博客迁移至github pages

终于把博客搭好了，哈哈。我使用github pages + hexo，果然很简单方便。
我认为使用它们搭建博客有以下几点好处:

  1. 不折腾，能主要把精力放在写作上。
  2. 有漂亮的主题可选。
  3. 能熟悉建站的一些过程（当然，比不上自己搭建一个网站）。
  4. 与我曾经使用的blogspot等相比，更加自由。

我的博客主要用来记录平日里的一些思考与想法，一些学术性、零碎的东西我一般使用Evernote进行记录，就不会把它放到博客上面来了。

在现在这样一个浮躁的时代，我把博客当作一个能够与自我对话，剖析自我，并实现自我成长的地方。这是我写博客的初衷。
希望自己能够不忘初心，方得始终吧。
